## Structure newsletter
1. Accroche : 2 phrases maximum, percutantes.
2. Contexte astro : ce qui se passe dans le ciel ce mois / cette saison.
3. Impact personnel : comment cela résonne avec le thème natal du destinataire.
4. Conseil pratique : une action concrète, réaliste, directement utilisable.
5. Clôture : courte, sans formule creuse.

Longueur cible : 250–400 mots. Ton : professionnel mais humain.
